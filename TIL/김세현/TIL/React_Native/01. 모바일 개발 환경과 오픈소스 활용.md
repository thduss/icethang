## 🔵 1. React Native란 무엇인가?
React Native는 Facebook이 개발한 오픈소스 프레임워크입니다. 

웹 개발에 사용되는 React와 동일한 문법과 컴포넌트 기반 구조를 사용하면서도, 최종 결과물은 웹이 아닌 **네이티브 모바일 애플리케이션**입니다.


### ◾ 1-1. 크로스 플랫폼 개발
- React Native의 가장 큰 장점은 **크로스 플랫폼(Cross-Platform)**개발을 지원한다는 것입니다. 

- 하나의 코드로 iOS와 안드로이드 앱을 동시에 만들 수 있습니다. 이는 개발 시간과 비용을 획기적으로 줄여줍니다.


### ◾ 1-2. 네이티브 컴포넌트
- React Native는 웹의 `div`나 `p` 태그 대신, **네이티브 UI 컴포넌트**(`View`, `Text` 등)를 사용합니다. 
- 이러한 컴포넌트들은 각 운영체제(iOS/안드로이드)의 네이티브 UI로 변환되어 실행되므로, 웹뷰 기반의 하이브리드 앱보다 훨씬 부드럽고 성능이 뛰어납니다.

## 🔵 2. React Native 개발 환경 구축
- React Native 개발 환경은 크게 두 가지 방식으로 구축할 수 있습니다. 
- 여기서는 **더 자유로운 개발이 가능한 React Native CLI를 기준**으로 설명하며, 안드로이드 SDK 설치 과정을 포함하겠습니다.
