## 🔵 1. React Native의 핵심 컴포넌트 이해하기
- React Native는 웹의 HTML 태그 대신, 모바일 앱을 구성하는 네이티브 컴포넌트들을 사용합니다.
- 러한 컴포넌트들은 각 플랫폼(iOS, Android)의 고유한 UI 요소로 변환되어 화면에 렌더링되므로, 웹뷰를 사용하는 하이브리드 앱보다 훨씬 뛰어난 성능과 사용자 경험을 제공합니다.

### 1. `View`
- View는 UI를 구축하는 가장 기본적인 블록.
- 웹의 `<div>`와 비슷하며, 다른 컴포넌트들을 담는 컨테이너 역할.
- 레이아웃과 스타일을 적용하는 데 주로 사용.
- View는 그 자체로는 눈에 보이지 않지만, 모든 UI 요소의 기반.
