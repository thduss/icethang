# 앱 화면 구성
## 📱 전체 앱 구조 한눈에 보기

```css
[ 선생님 앱 ]           [ 학생 앱 ]
Teacher App            Student App
────────────           ────────────
- 수업 제어             - 홈 (캐릭터)
- 사전 알림             - 수업 중 오버레이
- 행동 로그             - 보상/기록
```

👉 **교사 = 제어 & 판단**

👉 **학생 = 행동 & 피드백**

---

## 화면 flow

### 1️⃣ 학생 앱 화면 flow

```css
[ 앱 실행 ]
   ↓
**[ 학생 홈 ]**
- 캐릭터
- 오늘 목표 or 시간표
- 대기 상태
   ↓ (수업 시작 알림)
[ 수업 중 상태 ]
   ↓
[ 태블릿 사용 여부 판단 ]
   ├─ **태블릿 사용 O**
   │     ↓
   │  [ 미니 오버레이 ON ]
   │  - 🚦 신호등
   │  - ⭐ 보상 게이지
   │
   │     ├─ 🔴 집중 모드
   │     │    ├─ 정상 → 유지
   │     │    ├─ 들썩임 감지
   │     │    │     ↓
   │     │    │  [ 🟡 노란불 (약한 표시) ]
   │     │    │     └─ 계속 움직임 → 선생님 알림
   │     │    │
   │     │    └─ 화면 이탈 → 선생님 알림
   │     │
   │     └─ 🟢 자유 모드
   │          └─ 감지 OFF / 보상 없음
   │
   └─ **태블릿 사용 X**
         ↓
      [ 집중 전용 화면 ]
      - 캐릭터
      - 🚦 신호등
      - AI 안전벨트
         ├─ 🔴 집중 모드
         │    ├─ 정상 → 유지
         │    ├─ 들썩임 감지
         │    │     ↓
         │    │  [ 🟡 5·4·3·2·1 카운트 ]
         │    │     └─ 계속 움직임 → 경고 연출 + 선생님 알림
         │
         └─ 🟢 자유 모드
              └─ 감지 OFF
   ↓
[ 수업 종료 ]
   ↓
**[ 오늘 요약 & 보상 ]**
- 집중 성공 횟수
- 획득 보상
- 캐릭터 변화

```

### 2️⃣ 선생님 앱 화면 flow

```css
[ 앱 실행 ]
   ↓
**[ 오늘 수업 리스트 ]**
- 교시별 수업
- 현재 교시 강조
- 관리 중 학생 수 (결석, 출석 표시)
   ↓
[ 수업 시작 ]
(학생 앱에 수업 시작 신호 전송)
   ↓
**[ 수업 제어 화면 ]**
   ├─ 🚦 신호등 변경
   │    ├─ 🔴 집중
   │    └─ 🟢 자유
   │
   ├─ ⚠️ 이벤트 카드 수신 (필요 시)
   │
   └─ ⏱ 수업 타이머 표시
   ↓
[ 수업 종료 ]
(학생 앱에 수업 종료 신호 전송)
   ↓
**[ 요약 리포트 ]**
- 집중 성공 횟수
- 경고 발생 수
- 주요 이벤트 타임라인

```
