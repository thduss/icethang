## 시니어 프렌들리 회의
> **연령 다양성을 지원하는 비디오 컨퍼런스 플랫폼**

```
시니어 
=> 나이가 많아서 배려가 필요한 사람이 아니라, 환경이 배려하지 않아 불편을 겪는 사용자
```


고령 근로자가 증가함에 따라,
회의 참여 과정에서 발생하는 시각·청각·기술적 장벽을 낮추고
모든 연령이 동등하게 회의에 참여할 수 있도록 돕는 플랫폼입니다.

## 🎯 핵심 목표
- 회의 기능을 늘리는 것이 아니라
**회의에 참여하는 ‘부담’을 줄이는 것**

- 사용자가 설정을 공부하지 않아도
**자동으로 맞춰지는 회의 환경 제공**


## ✨ 핵심 기능 요약

### 1. 실시간 자막 & 요약 자막
- 실시간 발언 자막 제공
- 중요한 발언은 굵게 / 색상 강조
- 화면 하단에 “방금 요점” 한 줄 요약 표시

### 2. 회의 후 전화 요약
- 화면을 다시 보지 않아도 되는 회의 정리
- 회의 종료 후
  - 요약 내용을 문자 메시지로 전송
  - 또는 자동 음성 전화로 핵심 내용 전달


### 3. 개인 맞춤 난이도 자동 설정
> 사용자가 직접 설정하지 않아도 되는 환경
- 첫 사용 시 간단한 질문
  - “글씨가 잘 보이시나요?”
  - “말이 빠르다고 느끼시나요?”

- 응답 기반 자동 조정
  - 글씨 크기
  - 발언 재생 속도
  - 화면 버튼 개수


### 4. ‘이해 안 됨’ 버튼
> 말 끊지 않고 도움 요청
- 버튼 하나로 의사 표현
- 발언자에게만 조용히 표시
- 회의 후 요약에서 해당 발언 자동 강조


### 5. 가독성 강화 모드
- 큰 글씨 모드: 전체 텍스트 확대
- 가변 강조 모드:
  - 발언자 이름
  - 핵심 키워드
  - 주요 버튼만 크게 표시


### 6. 청각 보조 기능
- 특정 주파수 증폭으로 노인성 난청 보완
- 개인별 청각 특성에 맞춘 소리 보정


### 7. 개인별 발언 속도 조절
- 발언이 빠를 경우 **내 화면에서만 느리게 재생**
- 다른 참가자의 회의 흐름은 유지


### 8. 단순 UI 모드
- 복잡한 기능 숨김
- “참여하기” 버튼 중심의 화면 구성
- 필수 기능만 노출


### 9. 기술 도우미 연결
- 카메라·마이크 오류 감지
- 자동 가이드 제공
- 해결되지 않으면 **헬프데스크 연결


### 10. 음성 명령 제어
- 버튼 조작 없이 음성으로 기능 사용
  - “화면 공유 시작”
  - “손 들기”
  - “회의 나가기”


### 11. 사전 테스트 모드
> 회의 전 미리 확인하는 안전 장치
- 회의 30분 전 자동 알림
- 전용 테스트 방 제공
  - 카메라
  - 마이크
  - 자막
  - 음성 명령 작동 여부 확인


## 한 문장 정리
> 시니어 프렌들리 회의는
> 기술을 배우게 하는 플랫폼이 아니라,
> 기술이 먼저 사용자를 배려하는 회의 환경입니다.
