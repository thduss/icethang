### **실시간 팩트체크 회의란?**

회의 중에는 추측이나 부정확한 정보가

사실처럼 전달되는 경우가 자주 발생합니다.

이 프로젝트는 회의 도중 발언된 정보에 대해

**실시간으로 사실 여부를 확인하고,
정보의 신뢰도를 시각적으로 보여주는 회의 보조 플랫폼**입니다.

---

### 기획 배경

- 회의 중 수치·통계·사실이 정확한지 바로 확인하기 어려움
- “그건 나중에 확인해볼게요”로 넘어가는 경우가 많음
- 회의 후 검증 과정에서
    - 잘못된 정보가 이미 공유됨
    - 의사결정이 번복되는 문제 발생
- 기존 화상회의 도구는
**발언의 ‘사실 여부’에는 관여하지 않음**

---

### 핵심 컨셉

- 회의 내용을 판단하거나 통제 ❌
- 발언을 검열 ❌
- **정보의 신뢰도를 보조적으로 제공 ⭕**

---

### 핵심 기능

1. **발언 실시간 팩트체크**
    - 회의 중 발언된 내용 중
        - 수치
        - 날짜
        - 사실 주장
    - AI가 자동 감지하여 **내부 데이터베이스, 사전 등록된 회사 자료** 와 대조
    
2. **출처 추적 버튼**
    - “이 정보의 근거가 무엇인지 바로 확인”
    
3. **데이터 모순 경고**
    - 동일 회의 내에서
        - 수치
        - 주장
        - 사실이 서로 다를 경우
    - **불일치 알림** 제공
    
4. **외부 정보 실시간 검증**
    - 회의 중 언급된
        - 통계
        - 시장 데이터
        - 공공 정보
    - 신뢰 가능한 외부 출처를 기반으로 검증
    
5. **투명성 배지 시스템**
    - 발언 옆에 신뢰도 표시
        - ✅ **확인됨**: 데이터로 검증된 발언
        - ⚠️ **추정**: 근거 부족 또는 의견성 발언
        - ❓ **검증 필요**: 즉시 확인 어려운 정보

---

### 기존 회의 방식과의 차이

| 구분 | 기존 회의 | 실시간 팩트체크 회의 |
| --- | --- | --- |
| 정보 검증 | 회의 후 | 회의 중 |
| 사실/추정 구분 | 불명확 | 시각적 표시 |
| 출처 확인 | 별도 검색 | 버튼 클릭 |
| 오류 발견 | 늦음 | 즉시 |

---

### 기술 방향

- 음성 인식(STT) 기반 발언 텍스트화
- 핵심 주장 및 수치 자동 추출
- 내부 DB / 외부 데이터 연동
- 신뢰도 기준에 따른 배지 표시
- 회의 흐름을 방해하지 않는 UI 설계
